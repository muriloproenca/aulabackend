# Projeto Tarefa

- login
- cadastrar a tarefa
- concluir a tarefa
- modificar a tarefa
- excluir a tarefa
- status da tarefa
- data limite

## Requisitos

- Rodar web
- responsivel para rodar em celular
- as tarefas armazenadas em BD
- deve ter uma API

## Implementação

- backend em nodejs
- frontend em bootstrap + jquery
- BD mongodb
- hospedado em servidor externo
